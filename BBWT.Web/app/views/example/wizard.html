Generally, prefer tabs to wizards. However, if the users are expected to be inexperienced with the page over a long time, and three-quarters of the sections are mandatory, then consider wizards. For instance, it is reasonable for the company sign-up page to be a wizard because we expect it to be used by a user once and then never again.
