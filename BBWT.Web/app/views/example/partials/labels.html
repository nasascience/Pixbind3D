Labels and Inputs

We use the following:
  • Label: Dark grey, #474747
  • Input Text: Light grey, #6f6f6f
  • Disabled Text: Grey, #b5b5bb5
  • Space from end of prompt/label to start of element: 15px
  • Inner margin inside elements: 12px

The controls should be aligned at a distance from the left hand side equal to 1/6th of the container's width. The labels for the controls should be right-aligned.

The label takes the default width of 1/6th of the containter and the input the default width of 1/4th

<div class="form-horizontal">
    <div class="form-group">        
        <label for="textField" class="col-xs-10 col-sm-4 col-xs-offset-1 col-sm-offset-0 control-label">Label:</label>
        <div class="label-content col-xs-10 col-sm-4 col-xs-offset-0 col-sm-offset-0">
            <input name="textField" type="text" class="form-control" value="Text field" />
        </div>
    </div>
</div>

Multiline labels:

<div class="form-horizontal">
    <div class="form-group">
        <label for="textFieldWithLongLabel" class="col-xs-10 col-sm-4 control-label">This is an extraordinarily long label and long labels should be avoided wherever possible:</label>
        <div class="label-content col-xs-10 col-sm-4">
            <input name="textFieldWithLongLabel" type="text" class="form-control" placeholder="Text field with multiline label" />
        </div>
    </div>
</div>

On a page displaying read-only controls only, align the controls to the left. Try and avoid mixing editable and read-only controls on the same pane. Where this is not possible prefer showing all the controls centrally aligned.
