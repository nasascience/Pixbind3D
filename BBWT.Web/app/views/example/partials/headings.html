Headings

The heading above is an <h3> tag:

<h3>Headings</h3>

Here is an <h1> tag, which we use at the top of the page. If the page is for managing a list, adding an object, or editing an object then the page title should be ‘Manage <plural form of the objects in the list>’, ‘Add <singular of the object>’ or ‘Edit <name of the object>’ respectively, e.g. ‘Manage Companies’, ‘Add Company’, ‘Edit Blueberry Consultants Ltd’. We should always refer to an object by name in preference to an ID. If we don't have a name, then use the type of object followed by the ID, e.g. 'Edit Brake Part #23445'.

Page Title

<h1>Page Title</h1>

In some sites we want a sub-part of the heading, which indicates an object ID or similar. We do that with a <small> inside the h1 tag:

Edit Brake Part #23445

<h1>Brake Parts <small>serial #23445</small><h1>

Prefer tabs to header sections when the page will see lots of use by inexperienced users.
