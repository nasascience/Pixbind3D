Links

You may want to have links in a form.

A link 
<div class="form-horizontal">
    <div class="form-group">
        <label class="label-content col-xs-10 col-sm-4 control-label">Label:</label>
        <div class="col-xs-10 col-sm-4">
            <span class="form-control-static"><a href="">A link</a></span>
        </div>
    </div>
</div>

Anchor links (href="#!foo") need special care. You must set 'reloadOnSearch=false' on your route and the value of the href must include the full path even when linking within the page. Example:

Links section Widths section

HTML code for those links looks like: 
<a href="/example/guidelines#links">Links section</a>
<a href="/example/guidelines#widths">Widths section</a>
