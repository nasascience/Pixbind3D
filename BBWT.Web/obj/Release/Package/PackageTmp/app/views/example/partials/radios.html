Radio Buttons and Checkboxes

A set of radios each with their own label, to the right of the main label:

<div class="form-horizontal">
    <div class="form-group">
        <label class="col-xs-10 col-sm-4 control-label">Label:</label>
        <div class="label-content col-xs-10 col-sm-4 col-xs-offset-1 col-sm-offset-0">
            <div class="radio">
                <label>
                    <input type="radio" name="Radios1" id="radio3" value="option1" checked>
                    Label1
                </label>
            </div>
            <div class="radio">
                <label>
                    <input type="radio" name="Radios1" id="radio4" value="option2">
                    Label2
                </label>
            </div>
        </div>
    </div>
</div>

A set of radios with the options all on the same line:

<div class="form-horizontal">
    <div class="form-group">
        <label class="col-xs-10 col-sm-4 col-xs-offset-1 col-sm-offset-0 control-label">Label:</label>        
        <div class="label-content col-xs-10 col-sm-4 col-xs-offset-1 col-sm-offset-0">            
            <label class="checkbox-inline">
                <input type="radio" name="inlineRadios" id="Radio3" value="option1">
                Yes
            </label>
            <label class="checkbox-inline">
                <input type="radio" name="inlineRadios" id="Radio4" value="option2">
                No
            </label>
        </div>
    </div>
</div>

When in a vertical list of mixed controls, checkboxes should appear to the right of their labels. Where a checkbox is by itself, and especially if it requires a long sentence for its label then it consider sowing it on the left of its label.

HTML for this:

<div class="form-horizontal">
    <div class="form-group">        
        <label for="rightCheckbox" class="col-xs-4 col-sm-4 col-xs-offset-1 col-sm-offset-0 control-label">Checkbox:</label>
        <div class="label-content checkbox col-xs-1 col-sm-4">
            <input id="Checkbox1" type="checkbox" style="margin-left: -10px !important" />            
        </div>
    </div>    
    <div class="form-group">        
        <div class="col-sm-6 col-xs-offset-1 col-sm-offset-4">
            <label class="checkbox" style="margin-left: 5px">
                <input type="checkbox" />
                    This is a checkbox that requires a long sentence to describe its purpose 
            </label>
        </div>
    </div>
</div>
