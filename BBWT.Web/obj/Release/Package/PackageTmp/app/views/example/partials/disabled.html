Disabled Controls

If there's text in the textfield, 'Test button' will become disabled:

HTML:

<div class="form-horizontal">
    <div class="form-group">
        <label for="disabilngTextField" class="col-xs-10 col-sm-4 control-label">Label:</label>
        <div class="label-content col-xs-11 col-sm-4">
            <input name="disabilngTextField" type="text" class="form-control" ng-model="testModel.disablingText" />
        </div>
    </div>
    <div class="form-group">
        <div class="col-xs-11 col-sm-8">
            <button class="k-button pull-right" ng-disabled="testModel.disablingText">Test button</button>
        </div>
    </div>
</div>
