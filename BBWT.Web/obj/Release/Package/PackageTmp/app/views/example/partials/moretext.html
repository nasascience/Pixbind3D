Disabled Controls

To disable a control use the ng-disabled directive. This sets the disabled attribute on an element and adds the 'k-state-disabled' class. 

<div class="form-horizontal">
    <div class="form-group">
        <label for="disabledField" class="col-xs-10 col-sm-4 control-label">Label:</label>
        <div class="label-content col-xs-10 col-sm-4">
            <input name="disabledField" type="text" class="form-control" value="Disabled" ng-disabled="true" />
        </div>
    </div>    
</div>

If your field can never be enabled then it is a read-only field. Please see the Labels and Inputs section for read-only fields.
