Basic Page Structure

Here is the basic layout of a page:
Header
  • Menu

Page Title

A page description.
...etc...
  • Everything between the menu and footer has 20px space on all sides.
  • An optional page description under the page title.
  • Some space before the main content of the page.

HTML for basic page is:

<div class="page-header">
    <h1>Page Title</h1>
    <span class="purpose">A page description.</span>
</div>
<div class="form-horizontal">
    <div class="form-group">
        <label for="testText" class="col-sm-3 control-label">Label sample:</label>
        <div class="label-content col-sm-4">
            <input id="testText" class="form-control" type="text" />
        </div>
    </div>
    
    ...etc...

    <button class="k-button btn-pad-right">Save</button>
    <button class="k-button">Close</button>
</div>

Consider whether the page needs a description line. Description lines should be preferred when the use of a page is non-obvious, or when most of the user-base of the page will not be familiar with the page. Typically, Edit and Manage pages should not require a description line.
