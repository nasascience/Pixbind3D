Error Messages

We show an error message to the right of form fields that fail to meet criteria:

Please enter 'First name' 

<form name="testForm" novalidate class="form-horizontal">
    <div class="form-group">
        <label for="requiredField" class="col-xs-10 col-sm-4 control-label">First Name:</label>
        <div class="label-content col-xs-10 col-sm-4">
            <input name="requiredField" type="text" class="form-control" ng-model="test.requiredField" required />
        </div>
        <div class="col-xs-10 col-sm-4">
            <p class="form-control-static text-danger" ng-show="testForm.requiredField.$dirty && testForm.requiredField.$error.required">
                Please enter 'First name'
            </p>
        </div>
    </div>
</form>

An appropriate error message to display might be ‘Please enter a whole number’.

A mandatory field on a form that contains both mandatory and optional fields should be marked with a purple asterisk.

An appropriate error message to display for a mandatory field that was not completed is "Please enter '<label>'".
