Adjusting Label and Input Widths

You can alter the label widths using "col-sm-x" class. E.g. col-sm-5 sets the label 5/12 of container width. 

<div class="form-horizontal">
    <div class="form-group">
        <label for="col3" class="col-xs-10 col-sm-3 col-lg-3 control-label">Label:</label>
        <div class="col-xs-10 col-sm-3 col-lg-3">
            <input name="col3" type="text" class="form-control" value="col-sm-3 width" />
        </div>
    </div>
    <div class="form-group">
        <label for="col4" class="col-xs-10 col-sm-4 control-label">Label:</label>
        <div class="col-xs-10 col-sm-4">
            <input name="col4" type="text" class="form-control" value="col-sm-4 width" />
        </div>
    </div>
    <div class="form-group">
        <label for="col5" class="col-xs-10 col-sm-5 col-lg-5 control-label">Label:</label>
        <div class="col-xs-10 col-sm-5 col-lg-5">
            <input name="col5" type="text" class="form-control" value="col-sm-5 width" />
        </div>
    </div>
</div>
